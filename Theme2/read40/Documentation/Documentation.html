Getting Startedthings to know before get started

We assume you have basic knowledge of WordPress. In this doc you will learn how to setup, use and customize the theme. For information in regard to use the WordPress, please see the online manual of WordPress.

WordPress Settingsimportant

Administration > Settings:


General Settings: Site Title is plain text logo and Tagline is a slogan under the logo.

Permalink Settings: Post name option should be selected to work the theme properly.

Reading Settings: Select Your latest posts option if you want to use your blog posts as a homapage.

Installinstallation

Find the zipped theme file under Theme folder.

Go to Appearance > Themes > Install Themes > Upload.

Select your zipped theme file and install it. Activate the theme after installation completed.

Don't forget to create custom navigation menu, see the Menu section.

Dummy Datademo content

Navigate to Tools > Import > WordPress:

WordPress might offer you to install the WordPress importer plugin, if you haven't installed it before do it now.

Once you've installed WordPress importer plugin, browse to the folder Dummy Data in your download package, select dummy data file and upload it.

Go to Appearance > Menus > Theme Locations, select menu and click save button.

Menunavigation

This theme supports custom navigation menus.

Go to Appearance > Menus. Create your custom menu and save it from Theme Locations box.


Select Menu

Important: Don't forget to re-select and save your created menu from Theme Locations box after each theme updating.

Pagesdefault and custom page templates

Default Page Template: Has no sidebar.


Custom Page Templates

Select custom pages from template section of the Page Attributes box.


Page with Sidebar: Has a sidebar, define its sidebar from the Select Sidebar box.

Full width Page: Fluid page.

Archive: Shows your archives.

Gallery: This page is for your photo galleries to display them as thumbnails.

Portfolio: Use this page if have only one portfolio. But if you want to create many portfolios see the Multiple Portflio section.

Contact: To create a contact page with a contact form. After create go to Theme Options > Contact to define settings.

Homepage: A minimal homepage alternative which shows latest posts entries from the blog.

Homepagealternative

Create a page and select Homepage template for it. Then go to Settings > Reading. Select your homepage as a front page.

<audio width="300" height="32" style="width: 100%;" preload="none" src="http://themes.pixelwars.org/read-wp/wp-content/uploads/2013/01/AirReview-Landmarks-02-ChasingCorporate.mp3"></audio>

Portfolio Page

Create a new page and select Portfolio template from Page Attributes box.

Important: If you want to have seperate portfolio pages, go to Multiple Portfolios section.

Multiple Portfoliosmany separate portfolio pages

Important: Read carefully and try to understand how this functionality works.


You need a page with the template “Portfolio” if you want to use only one page portfolio on your site. But you don’t need any page for multiple portfolios on your site. Because the unlimited portfolio functionality doesn’t need any page to show portfolio items on your site. It works based on departments.

In multiple portfolios, every parent department works a different portfolio page.

Shortly:

Create parent departments. They will be your different portfolio pages. And create sub departments for your parent departments. These will be your portfolio filters.

Go to Appearance > Menus. Open the Departments box from the Screen Options panel.

Add your parent departments to your menu. These are portfolio pages.

Add new portfolio items to your sub departments.

Galleryphoto galleries

To create a gallery to show in the gallery page go to Gallery > Add New.

Widgetsdrag-drop contents

There are to many widget locations, sidebars and four columns in the footer.

Drag widgets to a location to use them. You can put one or more widgets to every widget location.

To create a new widget location ( Sidebar ) see the New Sidebar section.

New Sidebarnew widget location

This theme supports unlimited sidebars. You can create many sidebar and use them with the Page with Sidebar template ( see Pages > Page with Sidebar ).

Go to Appearance > Theme Options > Sidebar.

Enter your new sidebar name ( must be different from pre-created ). Click the Create button.

Go to Appearance > Widget. Your new sidebar will be displayed at the end of the widget location list on the rigt hand side column.

Drag and drop widgets to this new sidebar and create a page with Page with Sidebar template to use it.

Font Size Customizations

Text Logo Font Size

  1. Paste the code below to the Theme Options > Style > Custom CSS:
  2. h1.site-title { font-size: 5em; }
  3. Change the 5 value for different font sizes.

Tagline Font Size

  1. Paste the code below to the Theme Options > Style > Custom CSS:
  2. .site-description { font-size: 1.5em; }
  3. Change the 1.5 value for different font sizes.

Localizationtranslation

This theme is ready for localization. We have made every string translatable with the text domain "read", use Codestyling Localization plugin or Poedit editor.

The translation files are located under languages folder in the theme folder. ( Don't forget to copy after each theme updating. )


How to use Codestyling Localization plugin?

  1. Install and activate the plugin,
  2. Go to Tools > Localization > Themes,
  3. Select your language flag in the flag list that must be bold text or click Add New Language button to add it,
  4. Firstly click Rescan button and scan the theme files then click Edit button,
  5. All translatable strings will be displayed in the table,
  6. Every string has Edit link in the opposite of it to translate,
  7. Finally click generate-mo file button at the top side of the page.
